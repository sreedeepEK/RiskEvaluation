## RiskEvaluation 

This project focuses on analyzing insurance policy and claims data to identify risk patterns, claim drivers, and high-impact customer segments. 

### Preview 

![](https://github.com/sreedeepEK/RiskEvaluation/blob/main/image.png)

### Tools & Technologies

- SQL: Data querying, filtering, aggregations
- Power BI: Dashboards, DAX, data modeling
- Excel: Data validation and preprocessing

### Conclusion

This dashboard provides a comprehensive view of insurance risk and claim behavior. The insights can help insurers improve pricing strategies, risk segmentation, and claims management through data-driven analysis.
